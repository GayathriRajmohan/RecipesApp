# Recipes App in MERN Stack
An MERN stack app to view recipes together with preparation instructions, ingredients, and cooking times

## Getting Started

To run the app, simply run

```npm install yarn```

```yarn add expo```

```npx expo start```

And scan the QR code in the Expo client app or run in emulator using Android Studio.


Before running the app, make sure that the packages are installed in the correct versions as

- @babel/core@7.22.8
- @react-navigation/drawer@6.6.3
- @react-navigation/native@6.1.7
- @react-navigation/stack@6.3.17
- expo-status-bar@1.4.4
- expo@48.0.20
- react-native-gesture-handler@2.9.0
- react-native-reanimated@2.14.4
- react-native-safe-area-context@4.5.0
- react-native-snap-carousel@3.9.1 
- react-native@0.71.14
- react@18.2.0
- yarn@1.22.22

If not install using ```npm install package_name@version```
